# BTC-price
Get the current BTC price in deno

# How to use ?

## Use: 
`deno run -A --unstable --no-check https://deno.land/x/bitcoin@beta/exemple.ts`

## Or
See exemple.ts file

# Functions

## Get the price
```ts
await myvar.getPrice("Currency ?")
```

## Set currency
```ts
myvar.setCurreny("currency")
```
